
Overview
Build a full-stack web application from scratch and deploy it. The app fetches data from a public API, stores it in a database, and displays it on the frontend with a real-time search feature.

What to Build
React frontend — built from scratch (no templates)
Node.js + Express backend — built from scratch
When the app loads, the backend fetches posts from JSONPlaceholder API  →  jsonplaceholder.typicode.com/posts
Store those posts in MongoDB
Frontend calls the backend REST API and displays the posts
Search bar on the frontend uses WebSocket to query the backend in real time and show filtered results

APIs to Build
REST APIs
More than 1 required. At minimum: fetch & save posts, get all posts, get single post.

WebSocket API
Minimum 1 required. Used for the search feature — client sends a search query, server responds with matching posts.
